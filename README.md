# GRC Risk Management Project – FinTech Scenario

## Overview
This project simulates a Governance, Risk, and Compliance (GRC) program for a fictional FinTech company operating a cloud-based payment platform.

## Objectives
- Identify and assess cybersecurity risks
- Map controls to industry frameworks
- Perform a compliance gap assessment

## Frameworks Used
- NIST Cybersecurity Framework
- ISO 27001 (reference)

## Key Deliverables
- Risk Register with scoring methodology
- Information Security Policy
- Compliance Gap Assessment Report

## Tools Used
- Excel (Risk Register)
- Word (Documentation)
- Power BI (Dashboard)

## Key Highlights
- Identified 15+ risks including data breaches and insider threats
- Applied risk scoring (Likelihood × Impact)
- Recommended controls such as MFA, logging, and access reviews
